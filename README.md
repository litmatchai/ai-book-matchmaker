# 📚 LitMatch — AI Book Matchmaker

An AI-powered book recommendation app that matches you with your next 
favourite read based on your personality, mood, and interests.

 **Live Demo:** [ai-book-matchmaker.vercel.app](https://ai-book-matchmaker.vercel.app)

##  Features
- 📝 8–10 question personality quiz
- 🧠 6–7 unique reader personality types
- 📖 4–6 book recommendations per result (English & Arabic)
- 🌍 Full Arabic/English translation with RTL support
- 🔗 Direct PDF/read links for each book
- ⬅️ Go back and change answers mid-quiz
- ✨ Smooth animations and elegant book-themed UI

##  Built With
- React + Vite
- Tailwind CSS
- TypeScript
- Deployed on Vercel
